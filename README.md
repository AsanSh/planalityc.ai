# Planalityc.ai

Product prototype for a modular construction-company operating platform.

Core scenario:

1. Commercial director approves unit pricing.
2. Sales team sees only units opened for sale.
3. Contract creation generates accruals.
4. Payments close accruals and expose overdue debt.
5. Buyer portal and Client Relations module manage communication.

Run locally:

```bash
npm run dev
```

Build:

```bash
npm run build
```
